# 🤖 Mentoria com Agente de IA Generativa

## 📌 Sobre o Projeto

Este projeto tem como objetivo simular uma **mentoria personalizada de carreira em tecnologia**, utilizando um agente de IA generativa capaz de conduzir uma entrevista estruturada, analisar o perfil do usuário e sugerir caminhos profissionais adequados.

A proposta central é demonstrar como agentes inteligentes podem ser aplicados para **orientação de carreira**, combinando lógica de decisão, personalização e interação conversacional.

---

## 🎯 Objetivo

O projeto busca:

* Coletar informações relevantes do usuário por meio de uma entrevista guiada
* Identificar interesses, objetivos e nível de experiência
* Sugerir carreiras alinhadas ao perfil analisado
* Gerar um plano estruturado de desenvolvimento profissional
* Demonstrar o uso prático de agentes de IA generativa em um cenário real

---

## 🧠 Estrutura da Mentoria

A mentoria é dividida em duas etapas principais:

### 1. 🧩 Entrevista e Análise (Agent 1)

* Realiza perguntas estratégicas (uma por vez)
* Coleta dados sobre perfil, interesses e objetivos
* Analisa as respostas com base em critérios definidos
* Sugere carreiras mais adequadas ao usuário

### 2. 🚀 Planejamento de Carreira (Agent 2)

* Recebe os dados do perfil analisado
* Gera um roadmap completo de estudos
* Define habilidades, projetos e preparação para o mercado

---

## 📝 Documentação das Respostas

Todas as interações realizadas durante a mentoria foram **documentadas e estruturadas**, incluindo:

* Respostas do usuário para cada pergunta
* Interpretação do perfil
* Critérios de análise utilizados
* Justificativa das recomendações de carreira
* Plano de desenvolvimento sugerido

Essa documentação permite:

* Transparência no processo de decisão
* Reprodutibilidade da análise
* Uso como material de estudo ou portfólio

---

## 🛠️ Tecnologias e Conceitos Utilizados

* IA Generativa
* Engenharia de Prompt
* Modelagem de decisão baseada em regras
* Estruturação de agentes (multi-agent system)
* Personalização de respostas

---

## 💡 Aplicações Possíveis

Este projeto pode ser expandido para:

* Plataformas de orientação profissional
* Sistemas educacionais personalizados
* Mentorias automatizadas em larga escala
* Ferramentas de onboarding em empresas

---

## 📈 Conclusão

Este projeto demonstra como agentes de IA podem ir além de respostas simples, atuando como **mentores inteligentes**, capazes de analisar contexto, tomar decisões e guiar usuários em jornadas complexas — como a escolha e desenvolvimento de uma carreira em tecnologia.
